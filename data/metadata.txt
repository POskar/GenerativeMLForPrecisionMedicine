the file HAD.csv in this folder contains an anonymized version of data/HAD/HAD_extracted_features_v2.1_GSR+MRavailableonly.csv

the data from the original file is kept except for the "patid" column

a row shuffle was done to hide the original ordering of patients.

This data was originated at the Hadassah hospital in Jerusalem and is part of the EU-Validate project. 
It has been shared with Simula/OsloMet with permission from the hospital and the clinical model development work package leader at Validate.
